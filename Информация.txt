1. Описание:
Этот файл содержит информацию о скрипте, который
позволяет автоматически собирать информацию о пользователях из
выбранной Telegram-группы, включая их ID, имя пользователя и 
номер телефона (если доступен).

2. Предварительные требования:
- Вам понадобится аккаунт Telegram.
- Вам необходимо получить API ID и API Hash на портале разработчиков Telegram.
(Инструкция по получению api_id и api_hash - https://core.telegram.org/api/obtaining_api_id)

3. Правовая информация:
Используйте этот скрипт в соответствии с
политикой конфиденциальности Telegram, а также убедитесь,
что соблюдаете локальные законы и правила о защите персональных данных.
